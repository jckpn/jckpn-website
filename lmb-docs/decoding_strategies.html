Global Decoding Strategies

class BestOfN(n, stop_criteria_, objective_function=<lmblocks.utils.objective_functions.JointLogProbs object>, inner_call=None)

Generates N sequences and keeps the best one according to the objective function.

Parameters:

  • n (int) – Number of sequences to keep.

  • stop_criteria_ (StopCriterion) – Stop criterion for the decoding strategy. If None, the strategy will run for a single step.

  • objective_function (ObjectiveFunction) – Objective function to use for scoring sequences.

  • inner_call (DecodingStrategy) – Inner decoding strategy to use for generating sequences. Defaults to RandomSampling with the same stop criterion.

class BiasedTPS(steps, stop_criteria_, beta=1.0, objective_function=<lmblocks.utils.objective_functions.JointLogProbs object>, inner_call=None)

Biased TPS (transition path sampling) decoding strategy. Somewhat similar to the Metropolis-Hastings algorithm.

Parameters:

  • steps (int) – Number of repeats of the TPS step.

  • stop_criteria_ (StopCriterion) – Stop criterion for the decoding strategy. If None, the strategy will run for a single step.

  • beta (float) – The beta hyperparameter for the acceptance ratio (see ProbabilisticSequenceAccepter).

  • objective_function (ObjectiveFunction) – Objective function to use for scoring sequences.

  • inner_call (DecodingStrategy) – Inner decoding strategy to use for generating sequences. Defaults to RandomSampling with the same stop criterion.
