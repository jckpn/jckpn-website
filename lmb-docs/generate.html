Text and Sequence Generation

generate(input_text, model, decoding_strategy=<class 'lmblocks.decoding_strategies.local_strategies.GreedySearch'>, stop_criteria_=None, objective_function=<lmblocks.utils.objective_functions.JointLogProbs object>, num_completions=1, batch_size=32, return_type='str', seed=None, print_outputs=False, remove_input_text=False, output_path=None, mlflow_tracking_uri=None, mlflow_experiment_name=None, mlflow_objective_function=None)

Generates completions for the given input text using the given model and decoding strategy.

Parameters:

  • input_text (str | list[str]) – The input text to generate completions for. Can be a single string or a list of strings.

  • model (LanguageModel) – The language model to use for generation (see Models).

  • decoding_strategy (DecodingStrategy) – The decoding strategy to use for generation (see decoding_strategies). Defaults to GreedySearch.

  • stop_criteria – The stop criterion to use for generation. If not provided, the decoding strategy’s default stop criterion will be used.

  • objective_function (ObjectiveFunction | None) – The objective function to use for scoring sequences. Defaults to JointLogProbs(). Global strategies like BestOfN will use this function during generation.

  • num_completions (int) – The number of completions to generate for each input. Defaults to 1.

  • batch_size (int) – The batch size to use for generation. Defaults to 32.

  • return_type (Literal['str', 'SequenceBatch']) – The type of output to return. Can be either “str” or “SequenceBatch”. Defaults to “str”.

  • seed (int | None) – The random seed to use for generation. If not provided, the seed will be random.

  • silent – Whether to print progress information. Defaults to True.

  • remove_input_text (bool) – Whether to remove the input text from the output completions. Defaults to False.

  • output_path (str | None) – Any valid path to save the output completions to (must end in .jsonl).

  • mlflow_tracking_uri (str | None) – The URI of the MLflow tracking server.

  • mlflow_experiment_name (str | None) – The name of the MLflow experiment to log to.

  • mlflow_objective_function (ObjectiveFunction | None) – The objective function to use if logging to MLflow.

  • stop_criteria_ (StopCriterion | None)

  • print_outputs (bool | None)

Returns:

The generated completions as a string or a list of strings, or as a SequenceBatch object if specified.

Return type:

str | list[str] | SequenceBatch

Example 1:

from lmblocks import generate
from lmblocks.models import GPT2
from lmblocks.decoding_strategies import GreedySearch
from lmblocks.stop_criteria import MaxGeneratedTokens

out = generate("The sky is",
                model=GPT2(),
                decoding_strategy=GreedySearch(),
                stop_criteria_=MaxGeneratedTokens(5))
print(out)  # "The sky is the limit of time,"

Example 2:

out = generate(..., return_type="SequenceBatch")  # Same args as above
print(out)  # <SequenceBatch with 1 sequences>
print(out.to_text())  # ["The sky is the limit of time,"]
print(out.sequence_scores)  # torch.Tensor([-7.4193...])
